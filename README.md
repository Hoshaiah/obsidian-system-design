# Obsidian System Design

A structured note-taking system in Obsidian for studying and reasoning about system design.
Cards, dilemmas, and decision trees — connected as a graph.

## Structure

- `dilemmas/` — head-to-head tradeoff notes
- `patterns/` — reusable architectural patterns
- `case-studies/` — real-world systems broken down
- `templates/` — Obsidian templater files

## Why

System design knowledge isn't linear — it's a web. Obsidian's graph fits the shape.
